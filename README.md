# Pomodoro Timer ⏱️  
A clean, accessible, and customizable Pomodoro Timer built with **HTML, CSS, and JavaScript** as part of the [roadmap.sh](https://roadmap.sh/projects/pomodoro-timer) Frontend Projects.

This application helps users stay productive using the Pomodoro Technique — alternating focused work sessions with short and long breaks.
https://roadmap.sh/projects/pomodoro-timer
---

## 🚀 Features

### ✔ Start, Pause, Resume, Reset  
Full timer controls for smooth session management.

### ✔ Customizable Durations  
Users can configure:
- Work session (default: 25 minutes)  
- Short break (default: 5 minutes)  
- Long break (default: 15 minutes)

### ✔ Automatic Session Cycling  
- After each work session → short break  
- After 4 work sessions → long break  
- Tracks completed work sessions

### ✔ Audio Notification  
A sound plays when each session ends.

### ✔ Accessible & Responsive  
- Keyboard‑friendly  
- ARIA labels  
- Works beautifully on desktop and mobile

### ✔ Modular JavaScript  
State‑based logic with clean, reusable functions.
